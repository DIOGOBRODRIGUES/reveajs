O HTML DOM (modelo de objeto de documento)

Quando uma página da web é carregada, o navegador cria um modelo de objeto de documento da página. O modelo HTML DOM é construído como uma árvore de objetos :

árvore de objetos HTML DOM

Com o modelo de objeto, o JavaScript obtém todo o poder necessário para criar HTML dinâmico:

  • alterar e remover todos os elementos e atributos HTML da página
  • alterar todos os estilos CSS da página
  • adicionar novos elementos e atributos HTML
  • reagir a todos os eventos HTML existentes na página

O que é HTML DOM?

O HTML DOM é um modelo de objeto padrão e interface de programação para HTML. Ele define:

  • Os elementos HTML como objetos
  • As propriedades de todos os elementos HTML
  • Os métodos para acessar todos os elementos HTML
  • Os eventos para todos os elementos HTML

O método getElementById

A maneira mais comum de acessar um elemento HTML é usar o id do elemento.

A propriedade innerHTML

A maneira mais fácil de obter o conteúdo de um elemento é usando a propriedade innerHTML . A propriedade innerHTML é útil para obter ou substituir o conteúdo de elementos HTML.
