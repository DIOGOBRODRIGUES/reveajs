Introdução

React é uma biblioteca JavaScript para construção de interfaces de usuário. React é usado para construir aplicativos de página única. React nos permite criar componentes de UI reutilizáveis.

Criando aplicativo React

Para aprender e testar o React, você deve configurar um ambiente React em seu computador. É nescessário fazer instalação no NODE.JS e depois Execute este comando para criar um aplicativo React chamado my-react-app 

npx create-react-app my-react-app

Execute o aplicativo React

Execute este comando para ir para o my-react-appdiretório: 

cd my-react-app

Execute este comando para executar o aplicativo React my-react-app: 

npm start

O que é React?

React, às vezes chamado de framework JavaScript frontend, é uma biblioteca JavaScript criada pelo Facebook. React é uma ferramenta para construir componentes de UI.

Como funciona o React?

React cria um DOM VIRTUAL na memória. Em vez de manipular diretamente o DOM do navegador, o React cria um DOM virtual na memória, onde faz todas as manipulações necessárias, antes de fazer as alterações no DOM do navegador.

O React só muda o que precisa ser mudado!

O React descobre quais alterações foram feitas e altera apenas o que precisa ser alterado.
