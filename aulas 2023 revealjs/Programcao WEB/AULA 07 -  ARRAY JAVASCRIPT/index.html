Introdução

Um array é uma variável especial que pode conter mais de um valor: 


						const cars = ["Saab", "Volvo", "BMW"];

Acessando Elementos do Array

Você acessa um elemento do array referindo-se ao número do índice : 


						const cars = ["Saab", "Volvo", "BMW"];
						let car = cars[0];

Alterando um elemento da matriz

Esta instrução altera o valor do primeiro elemento em cars:


							cars[0] = "Opel";

Arrays são objetos

O HTML DOM é um modelo de objeto padrão e interface de programação para HTML. Ele define:

  • Arrays são um tipo especial de objetos. O operador typeof em JavaScript retorna “objeto” para arrays.
  • Os objetos usam nomes para acessar seus “membros”. Neste exemplo, person.firstName retorna John: 

						const person = {firstName:"John", lastName:"Doe", age:46};

Percorrendo um array

Uma maneira de percorrer um array é usando um loop for: 


						const fruits = ["Banana", "Orange", "Apple", "Mango"];
						let fLen = fruits.length;

    ";
						for (let i = 0; i < fLen; i++) {
							text += "

    • ";
						}
						text += "
