React Props

Props são argumentos passados ​​para componentes React por meio de atributos HTML.

Exemplo

Para enviar props para um componente, use a mesma sintaxe dos atributos HTML:

Adicione um atributo "marca" ao elemento Carro:


						const myElement = <Car brand="Ford" />;

Use o atributo brand no componente:


						function Car(props) {
							return 
I am a { props.brand }!
;
						  }

Exemplo


						import React from 'react';
						import ReactDOM from 'react-dom/client';

						function Car(props) {
						  return 
I am a { props.brand }!
;
						}

						const myElement = < Car brand="Ford" />;

						const root = ReactDOM.createRoot(document.getElementById('root'));
						root.render(myElement);

Exemplo

Se você tem uma variável para enviar, e não uma string, basta colocar o nome da variável entre chaves:


						function Car(props) {
							return 
I am a { props.brand }!
;
						  }
						  
						  function Garage() {
							const carName = "Ford";
							return (
							  <>
								
Who lives in my garage?

								<Car brand= { carName } />
							  </>
							);
						  }

Exemplo

Ou se fosse um objeto:


						function Car(props) {
							return 
I am a { props.brand.model }!
;
						  }
						  
						  function Garage() {
							const carInfo = { name: "Ford", model: "Mustang" };
							return (
							  <>
								
Who lives in my garage?

								<Car brand= { carInfo } />
							  </>
							);
						  }

React Events

Assim como os eventos HTML DOM, o React pode realizar ações com base nos eventos do usuário. React tem os mesmos eventos do HTML: clicar, alterar, passar o mouse etc.

Adicionando Eventos

Os eventos React são escritos na sintaxe camelCase: onClick em vez de onclick.

Os manipuladores de eventos React são escritos entre chaves: onClick={shoot} em vez de onClick="shoot()".

Exemplo

Coloque a função shoot dentro da componente Football:


						import React from 'react';
						import ReactDOM from 'react-dom/client';

						function Football() {
						  const shoot = () => {
						    alert("Great Shot!");
						  }

						  );
						}

						const root = ReactDOM.createRoot(document.getElementById('root'));
						root.render(<Football />);
